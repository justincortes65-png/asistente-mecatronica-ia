# Asistente Mecatrónica IA

Sistema multi-agente para diagnóstico inteligente de sistemas electrónicos y mecatrónicos.

## Características

- Diagnóstico de fallas electrónicas
- Arquitectura multi-agente
- Sistema RAG
- Integración con Ollama
- Evaluación con DeepEval
- Trazabilidad con Opik

## Estructura del proyecto

```textS
app/
├── agents/
├── rag/
├── llms/
└── ui/

data/
├── manuals/
└── datasheets/

tests/
```
